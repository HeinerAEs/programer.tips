Swapping variables

Like other programming languages, when swapping two variables, we often use a temporary one:

var a = 'programmer', b = 'tips', temp;
temp = a;
a = b;
b = temp;

But you can do it without using temporary variable as:

var a = 'programmer', b = 'tips';
b = [a, a = b][0];
console.log(a, b);  // --> tips programmer

Notes

ECMAScript 6 provides a shorter and more easy to understand way:

[a, b] = [b, a];

To swap two numbers, we can do:

var a = 6, b = 9;
a = b + (b=a, 0);
console.log(a, b);  // --> 9 6
