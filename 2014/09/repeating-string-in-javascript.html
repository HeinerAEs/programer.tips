Repeating string in Javascript

  • javascript
11 September 2014
Improve it on Github

How to generate a string which is repeated by given string in Javascript?

var multiplier = 5,
    input      = 'a',
    output     = new Array(multiplier + 1).join(input);
console.log(output);   // 'aaaaa'
